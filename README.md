Sistema de Gestión de Contenido Audiovisual

Información Académica


Materia: Programación Orientada a Objetos 
Universidad: Universidad Politécnica Salesiana
Semestre: 2do


Objetivo General

Desarrollar un sistema en Java que permita gestionar diferentes tipos de contenido audiovisual (películas, series, documentales, podcasts y trailers) aplicando los principios de la **Programación Orientada a Objetos**.

Objetivos Específicos

1. Aplicar herencia para crear una jerarquía de clases desde ContenidoAudiovisual
2. Usar encapsulamiento con atributos privados y métodos públicos (getters/setters)
3. Implementar asociaciones (agregación y composición) entre clases
4. Demostrar polimorfismo con el método mostrarDetalles() en cada subclase
5. Crear una clase principal (PruebaAudioVisual) que utilice todas las funcionalidades

Propósito del Proyecto

Este proyecto sirve como un registro personal de entretenimiento. Permite al usuario almacenar y visualizar información organizada de:
- Películas con sus actores
- Series de TV con sus temporadas
- Documentales con sus investigadores
- Podcasts
- Trailers
